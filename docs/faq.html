FMEasyCalendar: Frequently Asked Questions

What is EasyCalendar?
EasyCalendar is a solution that can be used to display event data (appointments, reminders, and anything date-based) that is stored in a FileMaker database in a calendar format. With EasyCalendar, you can view events by month, week, day, etc - and do so using an interface that users will be familiar with.

What do the calendars look like?
Click here to see examples of the calendars generated with EasyCalendar.

Can EasyCalendar be used with FileMaker Go and/or FileMaker WebDirect?
Yes. EasyCalendar uses only native FileMaker functionality, making it fully compatible with both FileMaker Go and FileMaker WebDirect.

Can EasyCalendar support business rules?
Yes. You can use apply detailed business rules, so that users see only those calendar events that you want them to see.

For example, you might want all employees to be able to see company-wide meetings and events, but only those employees in the HR department to be able to see sensitive events (such as interview dates and times, performance review meetings, and so on).

The "$$Event_Filter" setting can be used to add criteria to the SQL SELECT statement that is used to gather events from the source tables. You can also branch the settings in the FMEasyCalendar script to enforce business rules.

Does EasyCalendar require FileMaker Server?
No. You can use EasyCalendar in local solutions, as well as solutions being hosted using peer-to-peer networking. However, when used in solutions that are being hosted with FileMaker Server 13, EasyCalendar will automatically use FileMaker 13's "Perform Script on Server" script step, which optimizes the generation and refreshing of calendars.

Does EasyCalendar support drag and drop?
No.

Does EasyCalendar support customization?
Yes. The calendar is displayed using standard layout objects, so if you are familiar with setting up layouts, you should be able to customize the display as much as you would like. In addition, you can easily change the way that individual events are displayed on the calendar, and apply both text color and styles to them.

Does EasyCalendar support multiple event sources?
Yes. You can display multiple types of events on a single calendar by adjusting EasyCalendar's settings.

Does EasyCalendar support localization?
We are still testing this feature, but it looks like EasyCalendar will automatically "localize itself" based on a user's settings. So you should see translated text (month and day names) and local formats for dates and times.

How much will EasyCalendar cost?
EasyCalendar will be provided via an open source license.

Who developed EasyCalendar?
EasyCalendar was developed by Tim Dietrich, a database consultant that specializes in the FileMaker platform.
